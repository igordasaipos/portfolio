# Igor Pinheiro - Portfolio

Personal portfolio website built with Next.js 15, React 19, Tailwind CSS v4, and Motion. Features minimalism with delightful animated components powered by [Motion-Primitives](https://motion-primitives.com).

## Features

- Minimal one-page portfolio layout.
- Blog support with MDX.
- Responsive and accessible design.
- Easy to use
- [Motion-Primitives](https://motion-primitives.com) for animated components.

## Getting Started

For detailed setup instructions, refer to the [Installation Guide](./INSTALLATION.md).

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

Deploy to Vercel or any hosting platform that supports Next.js.
