# Portfolio

Projeto base para um portfolio pessoal usando Vite, React, TypeScript, Tailwind CSS e shadcn/ui.

## Requisitos

- Node.js 18+ (ou versão LTS mais recente)
- npm, pnpm ou yarn

## Instalação

```bash
# instalar dependências
npm install

# ambiente de desenvolvimento
npm run dev

# checar lint
npm run lint

# build de produção
npm run build
```

## Estrutura

- `src/` contém a aplicação React (componentes, estilos e utilitários).
- `tailwind.config.ts` e `postcss.config.js` configuram Tailwind CSS.
- `components.json` deixa o projeto pronto para importar componentes via shadcn/ui.

Depois de instalar as dependências, rode `npm run dev` para iniciar o servidor e comece a personalizar com suas informações.
